# 🌐 HTML Course Project 1

A beginner-friendly project repository for practicing and understanding **HTML fundamentals**.  
This project is part of the *HTML Course Tutorial Series* designed to help learners build a strong foundation in web development.

---

## 📖 Overview
This repository contains the first project in the HTML course series. It demonstrates:
- Basic HTML structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- Common tags like headings, paragraphs, lists, links, and images
- Semantic elements for better readability and accessibility

The goal is to provide a **hands-on learning experience** for beginners starting their journey in web development.

---

## 🛠 Features
- Learn **[HTML Basics](ca://s?q=Explain_HTML_basics)** and webpage structure  
- Practice **[Headings & Paragraphs](ca://s?q=HTML_headings_and_paragraphs)** for content organization  
- Create **[Lists](ca://s?q=HTML_lists_tutorial)** (ordered & unordered)  
- Add **[Links & Images](ca://s?q=HTML_links_and_images)** for navigation and visuals  
- Use **[Semantic Tags](ca://s?q=HTML_semantic_tags)** to improve accessibility and SEO
